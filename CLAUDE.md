# Configurações do Projeto (CLAUDE.md)

Este arquivo serve como guia de preferências e comandos para o projeto. Agentes de IA lerão este documento para alinhar seu comportamento às suas preferências de desenvolvimento.

## 🛠️ Comandos Úteis

- **Instalar Dependências:** `npm install`
- **Executar Servidor Local:** `npm run dev`
- **Gerar Build de Produção:** `npm run build`
- **Visualizar Pré-visualização do Build:** `npm run preview`

## 📋 Diretrizes de Desenvolvimento

- **Idioma de Comunicação:** Todo o desenvolvimento, explicações e comentários no código devem ser em **Português**.
- **Mensagens de Commit:** **Todas** as mensagens de commit (títulos e descrições) devem ser escritas obrigatoriamente em **Português**, seguindo o padrão *Conventional Commits* (ex: `feat: adiciona componente de entrevista`, `fix: corrige alinhamento de botoes`, `chore: atualiza dependencias`).
- **Padrão de Código:** Manter separação clara de estilos no arquivo `src/index.css` e utilizar componentes funcionais com hooks eficientes para máximo desempenho.
